# 🎉 校园招聘表格版网站已完成！

## ✨ 新版特点

### 📊 **表格形式展示**（参考腾讯文档）
- ✅ 类似Excel的表格布局
- ✅ 清晰的列表展示
- ✅ 支持导出CSV

### 🎯 **只收录校园招聘**
- ✅ 不包含社会招聘信息
- ✅ 不显示具体岗位名称
- ✅ 不显示薪资信息
- ✅ 只展示：公司名、行业、分类、投递入口

### 📋 **显示内容**
每家企业显示8项信息：
1. 序号
2. 公司名称
3. 行业
4. 分类（500强/独角兽）
5. 招聘状态（招聘中/即将开始/已结束）
6. 截止日期
7. 更新时间
8. 官方投递入口（一键跳转）

## 🛠️ 如何添加新企业

在 `campus_recruitment.html` 中找到 `const companies = [` 这一行，在数组末尾添加：

```javascript
// 在现有数据后面添加
{
    no: 51, // 序号递增
    name: "新公司名",
    industry: "行业", // 互联网/金融/制造等
    category: "fortune500", // 或 "unicorn"
    status: "active", // active/upcoming/ended
    deadline: "2026-05-30", // 截止日期
    update: "2026-02-26", // 更新日期
    url: "https://公司官网校招地址"
},
```

保存后刷新页面即可看到新企业！

### Q: 数据会过期吗？
